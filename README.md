**Todo’s Lah** is a high-performance, cross-platform task management application built with Flutter. Designed for efficiency and a seamless user experience, it serves as a core demonstration of my mobile development and UI/UX capabilities.

📍 **Live Demo:** [https://santhoshh-maax.github.io/todo-s-lah-v1/](https://santhoshh-maax.github.io/todo-s-lah-v1/)

---

## 🌟 Key Features
- **Intuitive Task Management:** Create, update, and track daily goals with a clean, modern interface.
- **Local Persistence:** Your data stays on your device for privacy and speed.
- **Cross-Platform Support:** Fully optimized for Android and scalable for iOS/Web.
- **High Performance:** Smooth 60fps animations and lightweight architecture.
